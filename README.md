YouTube Downloader

Modern GTK4 + Libadwaita tabanlı, performans odaklı bir YouTube indirme uygulaması.
Flatpak sandbox uyumlu, kararlı ve temiz bir masaüstü deneyimi sunar.

Özellikler
 • Video ve ses indirme (yt-dlp altyapısı)
 • Playlist desteği
 • Light / Dark tema otomatik uyumu
 • M4A ve Opus formatlarında kapak görseli (cover art) embed desteği
 • Opus remux ve metadata koruma
 • İptal durumunda yarım dosyaları otomatik temizleme
 • İndirilen dosyayı dosya yöneticisinde açma
 • Flatpak sandbox uyumlu yapı

 Kurulum
 Tek Satır Terminal Kurulumu (Önerilen)
 wget https://github.com/SENTOR-X/youtube-downloader/releases/latest/download/youtube-downloader.flatpak && flatpak install youtube-downloader.flatpak
Kurulum sonrası çalıştırmak için:
 flatpak run com.sentor.youtubedownloader

 GUI ile Kurulum
 1. Release sayfasından .flatpak dosyasını indir
 2. Dosyaya çift tıkla
 3. Yazılım Merkezi üzerinden kur

Teknoloji

 • Libadwaita
 • yt-dlp
